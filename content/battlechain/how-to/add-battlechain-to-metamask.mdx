## Overview

To interact with contracts on BattleChain, you need to add the network to your wallet. This guide covers MetaMask, but the same details work for any EVM-compatible wallet.

Use the **Mainnet / Testnet** toggle in the top bar to choose which network these instructions apply to.

## Add the network

Click the button below to add the selected network to MetaMask automatically:

<NetworkInfo />

If you prefer to add it manually, open MetaMask, click the network selector at the top, choose **Add Network** → **Add a network manually**, enter the details below, and click **Save**:

<Network title="Mainnet">

- **Network Name**: BattleChain
- **RPC URL**: `%%mainnet.rpc%%`
- **Chain ID**: `626`
- **Currency Symbol**: `ETH`
- **Block Explorer URL**: `https://explorer.mainnet.battlechain.com/`

</Network>
<Network title="Testnet">

- **Network Name**: BattleChain Testnet
- **RPC URL**: `%%testnet.rpc%%`
- **Chain ID**: `627`
- **Currency Symbol**: `ETH`
- **Block Explorer URL**: `https://explorer.testnet.battlechain.com/`

</Network>

<Check>
BattleChain is now available in your MetaMask network list.
</Check>

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Cannot connect to RPC | Ensure the RPC URL is entered exactly as shown above |
| Wrong chain ID | Remove the network and re-add with Chain ID `626` (mainnet) or `627` (testnet) |
| Transactions stuck | Try resetting your MetaMask account (Settings → Advanced → Reset Account) |
