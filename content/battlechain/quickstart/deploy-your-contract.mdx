## What You'll Need

You need a dedicated wallet for BattleChain testing — **do not use a wallet that holds real funds.**

### Fund Your Test Wallet

**1. Add BattleChain to your wallet**

<NetworkInfo network="testnet" />

**2. Get free Sepolia ETH** from the [Google Cloud faucet](https://cloud.google.com/application/web3/faucet/ethereum/sepolia) — 0.05 ETH is plenty.

**3. Bridge to BattleChain** at [portal.battlechain.com/bridge](https://portal.battlechain.com/bridge) — select Sepolia to BattleChain Testnet and bridge your ETH.

<Tab title="AI CLI (foundry)">

Import your private key into Foundry's encrypted keystore. This is a manual step — your private key should never be passed to any AI tool.

```bash
cast wallet import battlechain --interactive
```

Paste your private key and set an encryption password. The key is stored encrypted at `~/.foundry/keystores/battlechain` — your raw private key is never written to disk.

Verify it worked:

```bash
cast wallet list
```

You should see `battlechain` listed. From here on, you'll enter the keystore password when transactions need signing.

<Tab title="Manual (foundry)">


On testnet, the DAO moderator is a permissionless contract — approve your own request immediately:

```bash
cast send 0x6C2DFbdF0714FC8CE065039911758b2821818745 \
  "approveAttack(address)" $AGREEMENT_ADDRESS \
  --account battlechain \
  --rpc-url https://testnet.battlechain.com:3051 \
  --legacy
```

Verify the state:

```bash
just check-state
```

| Output | Meaning |
|--------|---------|
| `2` | `ATTACK_REQUESTED` — waiting for DAO |
| `3` | `UNDER_ATTACK` — approved, vault is open |

</Tab>

<Note>
On testnet, the `MockRegistryModerator` lets you approve your own request immediately. On mainnet, this is a real DAO governance action.
</Note>

## Troubleshooting

### Forge scripts failing

Add `--skip-simulation` to any failing `forge script` command. Forge's local gas estimation doesn't work reliably on BattleChain.

<Tab title="Manual (foundry)">

Send a replacement transaction to clear the stuck nonce:

```bash
cast send \
  --account battlechain \
  --rpc-url https://testnet.battlechain.com:3051 \
  --nonce <stuck-nonce> \
  --gas-price <higher-price-in-wei> \
  --legacy \
  --value 0 \
  0x0000000000000000000000000000000000000000
```

Find your pending nonce: `cast nonce --rpc-url https://testnet.battlechain.com:3051 <your-wallet-address>`

</Tab>

### Out-of-gas failures

If a `forge script` command fails with a vague error even with `--skip-simulation`, try adding `-g 300` to use 3x the estimated gas.
