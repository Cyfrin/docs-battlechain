## How It Works

BattleChain is a **pre-mainnet, post-testnet environment** with real funds. It fills a critical gap in the Web3 development lifecycle:

```
Audit → Deploy to BattleChain → Stress Test (Attack Mode) → Promote to Production → Deploy to Mainnet
```

1. **Protocols** deploy audited contracts with real liquidity
2. **Whitehats** legally attack contracts and earn bounties under Safe Harbor
3. **The DAO** moderates to prevent abuse (e.g. mainnet copycats)
4. Battle-tested contracts **promote to production** and deploy to mainnet with confidence
